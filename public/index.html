API Endpoints

Home

HTTP Methods Path Description
GET / Home page

Users

HTTP Methods Path Description
GET /users Retrieve all users
GET /users/:id Retrieve a specific user by ID
GET /users/email/:email Retrieve a specific user by email
POST /users/login Login for user
POST /users/register Register a new user
PATCH /users/update/:id Update user information by ID
DELETE /users/delete/:id Delete a user by ID

Items

HTTP Methods Path Description
GET /items Retrieve all items
GET /items/:id Retrieve a specific item by ID
POST /items Add a new item
PUT /items/:id Fully update an item by ID
DELETE /items/:id Delete an item by ID

Cart

HTTP Methods Path Description
GET /cart Retrieve all items in the cart
POST /cart Add an item to the cart
PUT /cart/:prodID Update the quantity of a specific item in the cart
DELETE /cart/:prodID Remove a specific item from the cart
DELETE /cart Clear all items from the cart
